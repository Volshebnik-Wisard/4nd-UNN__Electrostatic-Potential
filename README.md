# Моделирование распределения электростатического потенциала в двумерной области

### Постановка задачи
В цилиндрический металлический стакан, имеющий радиус R и высоту H. Оси стакана и стержня совпадают. Корпус стакана заземлён, а на стержне имеется заряд Q. Рассчитать поле в зависимости от глубины погружения стержня. Нарисовать эквипотенциальные и силовые линии. Определить ёмкость в зависимости от глубины погружения стержня.

## **Среда разработки**
- Visual Studio 2022 с компонентами:
  - "Разработка классических приложений на C++"
  - "Пакет SDK для Windows 10/11"
  - "MSVC версии 141 - средства сборки C++"
  - "MSVC версии 141 - библиотеки C++"
